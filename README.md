# 🧠 Intelligent Symptom-Based Disease Prediction & Clinical Support System

This project was developed during my **Data Science Internship at Global Quest Technologies (GQT)**.

## 🚀 Overview

An interactive machine learning system that predicts diseases based on user symptoms and provides clinical decision support.

## 🔍 Features

* 🤖 Chatbot-style symptom input
* 📊 Confidence score & top predictions
* ⚠️ Severity analysis
* 💡 Personalized recommendations
* 🎨 Interactive UI using Streamlit

## 🧠 Tech Stack

* Python
* Machine Learning (Random Forest)
* Streamlit
* Pandas, Scikit-learn

## ▶️ How to Run

```bash
pip install streamlit pandas scikit-learn
streamlit run app.py
```

This system is for educational purposes only and not a substitute for professional medical advice.
